## Installation

1. Clone or [download][download] this repository
2. Navigate to the project folder with a CLI
3. Run the following:

   ```bash
   $ npm install
   $ cd src/
   $ npm install
   $ cd ..
   ```

## Development

To run the application directly in your development environment run:

```bash
$ npm run dev
```
